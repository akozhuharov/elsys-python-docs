# Задачи в клас
## Прочетете файл и изкарайте на екрана всички нечетни редове и ги запишете в втори файл

```python
with open("test.txt") as f:
    lines = f.readlines()

output = []
for index, line in enumerate(lines):
    if index % 2 == 0:
        continue
    print(line)
    output.append(line)
with open("output.txt", "w") as f:
    f.writelines(output)
```
